# E-Book Reader

A Java Swing desktop application for reading and managing
PDF books with user authentication and MySQL database storage.

## Features
- User login and registration system
- Add PDF books from your computer
- View all books in a clean dashboard
- Search books by title or author
- Mark books as Read or Unread
- Delete books from your library
- Stats showing total, read, and unread books
- Clean modern two-panel UI design

## Tech Used
- Java (JDK 8+)
- Java Swing (GUI)
- MySQL (Database)
- XAMPP (Local server)
- NetBeans IDE

## Requirements
- Java JDK 8 or higher
- XAMPP installed and running
- MySQL Connector JAR file
- NetBeans IDE

## Database Setup
1. Open XAMPP and start Apache and MySQL
2. Go to http://localhost/phpmyadmin
3. Create a new database called `ebookreader`
4. Run the SQL file provided in the `sql/` folder

## How to Run
1. Clone this repository
2. Open NetBeans → File → Open Project
3. Select the EBookReader folder
4. Add MySQL connector JAR to Libraries
5. Start XAMPP (Apache + MySQL)
6. Run the SQL setup script in phpMyAdmin
7. Press F6 to run the project
